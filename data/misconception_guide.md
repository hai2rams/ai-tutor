## Fractions

Students often add fractions by adding numerators and denominators directly.
Example error: 1/2 + 1/4 = 2/6.
Correct approach: find a common denominator first, then add numerators.
For 1/2 + 1/4, rewrite 1/2 as 2/4, then add to get 3/4.

Another common error is comparing fractions by denominator size only.
Remind students to compare the actual value, often by using common denominators or benchmarks like 1/2.
